# Default Node.js App

## 概述

Default TEN Framework application template written in Node.js

## 特性

- 基於 TEN Framework 建構
- 版本 0.10.18
- 跨平台支援

## 快速開始

### 前置條件

- TEN Runtime（版本 0.10.18 或相容版本）
- manifest.json 中指定的必需相依性

### 安裝

遵循 TEN Framework 套件安裝指南。

## 使用方法

此套件可以根據框架規範整合到 TEN 應用程式中。

## 授權

此套件是 TEN Framework 專案的一部分。
